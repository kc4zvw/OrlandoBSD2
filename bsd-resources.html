Derivatives

There are a number of software distributions based on FreeBSD. Notable derivatives include:

  • DesktopBSD — desktop-oriented operating system, originally based on KDE
  • FreeNAS — network-attached storage devices
  • FreeSBIE — live CD
  • FuryBSD — is a brand new, open source FreeBSD desktop
  • GhostBSD — MATE-based distribution, which also offers other desktop environments
  • HardenedBSD — is a security-enhanced fork of FreeBSD
  • MidnightBSD — is a BSD-derived operating system developed with desktop users in mind
  • NanoBSD — creates a FreeBSD system image for embedded applications
  • nomadBSD — a persistent live system for USB flash drives, based on FreeBSD®
  • PicoBSD — discontinued?
  • IntelliStar — satellite system that runs TV programs such as Weatherscan and Local On The 8s
  • m0n0wall — discontinued firewall
  • t1n1wall — A fork of m0n0wall, t1n1wall continues as a freebsd based firewal
  • OpenServer 10 — Xinuos server
  • OPNsense — is open source, FreeBSD-based firewall and routing software developed by Deciso
  • pfSense — (firewall)
  • TrueOS — previously known as PC-BSD and now discontinued
  • TrustedBSD Project — is an open source community developing advanced security features for the FreeBSD operating system
  • XigmaNAS — for network-attached storage devices
